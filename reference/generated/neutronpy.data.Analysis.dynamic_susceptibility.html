neutronpy.data.Analysis.dynamic_susceptibility¶

Analysis.dynamic_susceptibility(material, ei)[source]¶

Returns the dynamic susceptibility \(\chi^{\prime\prime}(\mathbf{Q},\hbar\omega)\)

Parameters:
material : object

Definition of the material given by the Material class

ei : float

Incident energy in meV
Returns:
counts : ndarray

The detector counts turned into the scattering function multiplied by the detailed balance factor
